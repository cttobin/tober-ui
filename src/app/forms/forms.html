Custom Icons

Customise the checkbox icons with CSS. For example, get a Font Awesome unicode definition from the Font Awesome cheat sheet and use that in your stylesheet:

.tob-checkbox {

  input[type="checkbox"] + label:before {

    // Unicode of icon when unchecked.
    content: '\f1db';

  }

  input[type="checkbox"]:checked + label:before {

    // Unicode of icon when checked.
    content: '\f058';
  }

}
